# 二次型的表示
1. $A^{T}=A$
2. $A$的主对角元素为平方项系数
3. $A$的其他元素为对应交叉项系数的一半
4. $二次型f的秩=r(A)$
# 化二次型为标准型
只含有二次项的叫做标准型
## 配方法
1. 先配所有含$x_1$的项
2. 再配所有含$x_2$的项
3. 当所有项都为完全平方项时，换元

## 正交变化法
1. 写出$f$的矩阵$A$
2. 求$A$的特征值$\lambda_1,\lambda_2,\lambda_3$
3. 求$A$的特征向量$\alpha_1,\alpha_2,\alpha_3$
4. 把中不正交的向量施密特正交化
5. 把以上所有向量单位化为$\xi_1,\xi_2,\xi_3$，则$Q=(\xi_1,\xi_2,\xi_3)$为正交矩阵
6. 作变换：$令x=Qy，可使f=\lambda_1y_1^2+\lambda_2y_2^2+\lambda_3y_3^2标准形$

# 正定二次型、正定矩阵
二次型$f$正定（即矩阵$A$正定）的判定方法
1. $A$的特征值全大于$0$（即$f的正惯性指数=未知数个数$）
2. $A$的各阶顺序主子式全大于0
