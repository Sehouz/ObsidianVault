# 效用理论论述
### 总效用和边际效用
- 基数效用论
	以效用基数衡量为基础而建立起来的效用理论被称为基数效用论
	局限：
	- 商品的效用涉及消费者对商品满足其欲望的心理评价，很难准确加以衡量
	- 对于解释消费者行为，效用的基数不是必要的，只需要确定实现目标的商品数量
- 序数效用论
	不以基数衡量效用满足程度，而是对不同商品组合按效用满足高低进行排序
	- 序数效用值/效用指数：用来表示消费者从不同商品中所获效用的**顺序**关系
	- 避免对效用的基数测度假设而又不改变基数效用论的基本思想

- 总效用（$TU$）：在一定时期内，消费者从消费商品获服务中获得的效用满足总量，取决于消费商品的总量$$TU=U(Q)$$
- 边际效用（$MU$）：在一定时期内，消费者每增加一单位某种物品消费所得的效用增加量，商品消费数量不同，编辑效用也不同$$MU=\frac{\Delta TU}{\Delta Q}=MU(Q)$$
### 边际效用递减规律
在一定时期内，其他条件不变狭，随着消费量增加，没增加一单位该商品或服务的消费中所获得的效用增加量是逐渐递减的
$$
\begin{align}
MU_x&=\frac{dTU_x}{dx} \gt 0\\
\frac{dMU}{dx}&=\frac{d^2TU}{dx^2} \lt 0
\end{align}
$$
货币作为特殊的商品，边际效用虽然服从递减规律，但是递减速度较慢，常记为常数：$MU_m=\lambda$
### 效用最大化
- 消费者效用最大化：如果其他条件不变，消费者在既定收入约束条件下实现了**最大化效用满足**，并保持这种情况不变，那么此时就称消费者处于均衡状态，简称为**消费者均衡**
#### 只消费一种商品
- 均衡条件：$MU=\lambda P$或$\frac{MU}{P}=\lambda$

#### 消费两种商品
- 效用函数：两种商品的组合可以给消费者带来效用，所以是两种消费了的一个函数：$$TU=U(Q_1,Q_2)$$
- 边际效用：假定另一种商品消费量保持不变，两种商品边际效用为：$$
\begin{align}
MU_1&=\frac{\Delta TU_1}{\Delta Q_1}=MU_1(Q_1,Q_2)\\
MU_2&=\frac{\Delta TU_2}{\Delta Q_2}=MU_1(Q_1,Q_2)
\end{align}
$$
	> ~~其实就是求对TU的两个偏导~~

- 限制条件：$P_1Q_1+P_2Q_2=m$，即消费者用于两种商品的支出之和等于总收入
- 均衡条件：在既定的收入约束条件下，消费者购买每一种商品获得的边际效用与其价格之比均相等，并且都等于货币的边际效用$$
\begin{align}
\frac{MU_1}{P_1}=\frac{MU_2}{P_2}&=\lambda \\
P_1Q_1+P_2Q_2&=m
\end{align}
$$

### 消费者剩余
- 消费者为得到一定数量的某种商品**愿意支付**的数额与**实际必须支付**的数额之间的差被称为消费者剩余

# 无差异曲线
## 偏好和选择
- 消费者偏好：消费者对商品（组合）的喜好程度。基本假设：
	1. 明确判断：消费者对任意两个商品组合都能进行排序
	2. 多个组合：消费者偏好满足传递性
	3. 未达充分满足：在其他商品数量相同的条件下，消费者更偏好数量大的商品组合
	4. 不同商品组合更优：消费者偏好多样性的商品组合
## 无差异曲线
- 无差异曲线：在特定偏好条件下，可以给消费者带来同样满足程度的商品不同数量组合描绘出来的曲线
	特征：
	- 有无数条。**离原点越远**，数量组合越大，代表的**效用水平越高**
	- 任意两条曲线都**不相交**
	- 一般**向右下方倾斜**
	- 大多数情况下**凸向原点**
## 边际替代率
- 边际替代率：两种商品间的替代程度可由商品的边际替代率来衡量。在**效用满足程度不变**条件下，增加一单位一种商品的消费可以替代的另一种商品的消费数量，简称**边际替代率**
	商品X对商品Y的边际替代率：$$MRS_{XY}=-\frac{\Delta Q_Y}{\Delta Q_X}|_{U不变}（即无差异曲线斜率绝对值）$$
- 边际替代率递减规律：保持效用水平不变条件下，随一种商品消费数量的增加，增加一单位该商品的消费而放弃另一种商品的消费数量逐渐减少
## 无差异曲线特例
- 完全互补品：L型曲线，边际替代率一边无穷大一边为0
- 完全替代品：向下倾斜的直线，边际替代率为常数

# 预算约束线
## 预算约束线
- 预算约束线：收入和商品价格既定的条件下，消费者全部收入能够买到的各种商品不同数量的组合
- 约束方程：$$P_1Q_1+P_2Q_2=m$$
## 预算约束线的变动
- 预算约束线的变动：当消费者收入和商品价格发生变动时，约束线也会随之变动
- 情形：
	1. 收入$m$发生变动：直线平移，增加向右
	2. 商品价格$P_1$变动：对应截距变动
	3. 商品价格同时变动：平移或旋转

# 消费者均衡
## 消费者均衡的决定
在收入和商品价格既定条件下，当消费者选择了最优消费数量，此时处于均衡状态
均衡点条件：
1. 位于预算约束线上
2. 改预算约束线与商品组合点所在的无差异曲线相切，即**斜率相同**
消费者均衡条件：两种商品的**边际替代率**等于这两种商品的**价格**之比：$$
\begin{align}
&MRS_{1,2}=-\frac{\Delta Q_2}{\Delta Q_1}=\frac{MU_1}{MU_2}; |k|=\frac{P_1}{P_2} \\
&MRS_{1,2}=|k|=\frac{MU_1}{MU_2}=\frac{P_1}{P_2} \\
&\frac{MU_1}{P_1}=\frac{MU_2}{P_2}
\end{align}
$$
> 这与技术效用理论给出的均衡条件时一致的：$\frac{MU_1}{P_1}=\frac{MU_2}{P_2}=\lambda$
## 收入变动对消费者均衡的影响
### 收入-消费扩展线
假定两种商品价格不变，随着消费者收入的变动，消费者均衡点变动的轨迹被称为**消费者的收入-消费扩展线**，简称收入扩展线。代表消费者其扩张消费的最优路径
- 正常品-正常品：收入扩展线是从原点出发向上的直线
- 奢侈品-必需品：原点出发越来越陡峭的曲线
- 正常品-低档品：先增后减，向正常品轴弯曲
## 价格变动对消费者均衡的影响
### 价格-消费扩展线
假定收入和其他商品价格不变，只有一种商品价格发生改变，消费者均衡点变动的曲线，简称价格扩展线。

# 价格变动的替代效应和收入效应
## 含义
收入和其他商品价格不变的条件下，一种商品价格的变动会对需求量产生影响，称为**价格变动的总效应**，可以分解为**替代效应**和**收入效应**两个方面
- 替代效应：**两种商品价格发生相对变动**，导致消费者在维持**原有效用水平不变**的条件下对该商品需求量做出调整
- 收入效应：某种商品价格变动引起消费者**实际收入水平变动**，导致消费者在保持价格不变的条件下对该商品需求量做出调整
## 消费者需求曲线形状
取决于该商品价格变动总效应的大小和方向

| 商品类型  | 替代效应 | 收入效应 | 总效应 | 需求曲线形状     |
| ----- | ---- | ---- | --- | ---------- |
| 正常品   | +    | +    | +   | 向右下方倾斜     |
| 普通低档品 | +    | -    | +   | 向右下方倾斜     |
| 吉芬商品  | +    | -    | -   | **向右上方倾斜** |
