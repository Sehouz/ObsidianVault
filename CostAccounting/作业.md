	3. 按生产步骤计算产品成本的方法

### 课堂练习
1. 采用平行结转分步法，在完工产品与在产品之间分配费用，是指在（C）两者之间的费用分配
	1. 产成品与月末在产品
	2. 完工半成品与月末加工中的在产品
	3. 最后步骤的产成品与广义的在产品
	4. 前面步骤的完工半成品与加工中的在产品，最后步骤的产成品与加工中的在产品
2. 采用平行结转分步法，完工产品是指（A）
	1. 企业最后完工的产品
	2. 广义的在产品
	3. 第一步骤的半成品
	4. 各步骤所消耗上一步骤的半成品
3. 广义的在产品包括（ABCD）
	1. 尚在本步骤加工中的在产品
	2. 转入各半成品库的半成品
	3. 全部加工中的在产品和半成品
	4. 已从半成品库转到以后各步骤进一步加工，尚未制成的半成品
4. 平行结转分步法，其特点是（ABCD）
	1. 各步骤可以同时计算产品成本
	2. 不能提供半成品成本资料
	3. 费用结转与半成品实物转移脱节
	4. 不能全面反映各个生产步骤产品的生产耗费水平
5. 判断题
	1. 产品成本计算的分步法，是按照产品的生产步骤归集生产费用，计算产品成本，它主要适用于大量、大批的单步骤生产。X
	2. 分步法的成本计算对象是各种产品的生产步骤和产品品种。$\checkmark$
	3. 在分步法下，如果生产多种产品，产品成本明细账应该按照每种产品的各个步骤开立。$\checkmark$
	4. 由于各个企业生产工艺过程的特点和成本管理对各步骤成本资料的要求不同，分步法可分为综合结转分步法和平行结转分步法两种。X
	5. 逐步结转分步法就是为了计算半成品成本而采用的一种分步法，因此也称其为计列半成品成本分步法。$\checkmark$
	6. 在分步法下，如果半成品完工后，通过半成品库收发，则应编制结转半成品成本的会计分录。$\checkmark$
	7. 逐步结转分步法实际上就是品种法的多次连续应用。$\checkmark$
	8. 成本还原的对象是产成品成本。X
	9. 不论是综合结转还是分项结转，半成品成本都是随着半成品实物的转移而结转。$\checkmark$
	10. 采用平行结转分步法，各生产步骤不计算半成品成本。$\checkmark$
	11. 成本还原，一般是按本月所产半成品的成本结构进行还原。$\checkmark$
	12. 逐步结转分步法只适宜在半成品品种不多、逐步结转半成品成本的工作量不是很大的情况下采用。X
